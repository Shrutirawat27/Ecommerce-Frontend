# E-Commerce Website
A fully functional e-commerce web application that includes product listing, cart management, user authentication, and admin-level features. Built using MERN stack with a modern UI, responsive layout, and secure backend APIs.

## Live Demo
https://herstyle-ecommerce.vercel.app/

## Features
- User registration and login
- Product listing with detailed view
- Add to cart and remove from cart
- Order placement and checkout flow
- Admin features for product and user management
- Responsive UI with clean styling
- Secure backend routes and validations
